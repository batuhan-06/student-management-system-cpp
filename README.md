# Student Management System (C++)

A console-based Student Management System built with C++ to manage student records using Object-Oriented Programming, STL, and file handling.

## Features

* Add student
* Show students
* Delete student
* Update student
* Save students to file
* Load students from file

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL (vector)
* File Handling (fstream)
* String Parsing (stringstream)

## Project Structure

* Student.h → class declaration
* Student.cpp → class implementation
* main.cpp → main logic and menu system

## Purpose

This project was built to strengthen my C++ fundamentals, data structures knowledge, file handling, and object-oriented programming skills.

## Example Data Format

Students are stored in a text file in the following format:

```text
1,Ali,20,Computer Engineering
2,Ayse,21,Software Engineering
```
